# HAProxy + Nginx Homework

## Цель задания
- Настройка балансировки нагрузки с помощью HAProxy
- Настройка связки HAProxy + Nginx

## Структура проекта
- `configs/` - конфигурационные файлы
- `scripts/` - скрипты установки и настройки
- `docs/` - документация и проверка работы

## Выполнение
1. Установка и настройка HAProxy
2. Настройка Nginx серверов как бэкендов
3. Тестирование балансировки нагрузки
